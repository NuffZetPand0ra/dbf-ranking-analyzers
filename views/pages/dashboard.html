Spilleranalyse

Handicap­sammenligning

Sammenlign spilleres handicapudvikling over tid. Upload individuelle spillerfiler fra bridge.dk og visualiser tendenser, identificer klubbernes topspillere og track progress over længere perioder.

  • Sammenlign op til 9 spillere på ét diagram
  • Filtrer på dato, tidsperiode (dag / uge / måned / kvartal / år)
  • Gennemsnitshandicap, minimums- og maksimumshandicap og ændring per spiller
  • Se udvikling per klub ved at sammenligne klubbens spillere
  • Justerbar kurveglattning for klarere trends

Klubsammenligning

Handicap­fordeling

Analysér handicapfordelingen på tværs af klubber og sammenlign klubbernes handicapkarakteristika. Upload HACAlle-filen og udforsk detaljerede statistikker per klub.

  • Fuldt histogram over DBf's handicapspænd
  • Filtrer ned til enkelt klub for detaljeret analyse
  • Sammenlign klubbers spillerfordeling og medianhandicap
  • Statistik per klub: spredning, median, min/max, antal spillere
  • 24-timers lokal cache — ingen genupload nødvendig

Spiller­badge

Generér et delbart badge for én spiller med HC-historik, fremtidsprognose og nøglestatistik. Se om din faktiske udvikling følger prognosen over tid.

  • HC-historik med justerbar datoperiode
  • Fremtidsprognose via lineær regression med optimisme-slider
  • Stabilitetscore og national/klub-percentil
  • Delebart via link — prognosen er fast-forankret i tid

Hvis Bare

If-Only­analyzer

Se din HC-udvikling som en alternativ delta-bane. Første version viser, hvordan kurven ville have set ud, hvis en valgt klubs posteringer ikke blev anvendt.

  • Genafspiller hele HC-historikken fra start ud fra relative deltas
  • Ekskludér én klub ad gangen og sammenlign med den faktiske kurve
  • Viser samlet effekt nu, antal ekskluderede posteringer og ekskluderet Δhc
  • Arkitekturen er klar til senere “hvis bare uden denne spiller”-scenarier
